# 类数组

* 实参列表是类数组

1. 属性要有索引(数字)属性
2. 必须要有length
3. 最好有push

```js
var obj = {
    "0" : 'a',
    "1" : 'b',
    "2" : 'c',
    "length" : 3,
    "push" : Array.prototype.push,
    "splice" : Array.prototype.splice(加上splice方法之后，这个对象的表现形式就和数组更加像了)
 }

 ```

* 类数组的好处: 既拥有数组的特性(拥有数组的某些方法)，又拥有对象的特性(可以有属性和方法)，所以存储数据的功能更加强大。
