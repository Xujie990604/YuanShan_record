# 默认子路由

## 使用匹配空

```js
{
    path: '/shop',
    component: () => import('../views/shop/Shop.vue'),
    children: [
      {
        path: '',  // 设置空路由为goods，也就是把默认子路由设置为goods
        name: 'goods',
        component: () => import('../views/shop/Goods')
      },
      {
        path: 'comments',
        name: 'comments',
        component: () => import('../views/shop/Comments')
      },
      {
        path: 'seller',
        name: 'seller',
        component: () => import('../views/shop/Seller')
      }
    ]
  }
```

## 重定向

```js
{
    path: '/user/:id',
    component: () => import('../views/User.vue'),
    // 两种方式都可以定义成功，默认子路由。
    1. // redirect: '/user/:id/posts',
    children: [
    2. {
        path: '', // 此时的''空路由就相当于/user/:id
        redirect: 'posts'  //将路由 /user/:id 重定向到 /user/:id/posts
      },
      {
        path: 'profile',
        name: 'profile',
        component: () => import("../views/user/Profile.vue")
      },
      {
        path: 'posts',
        name: 'posts',
        component: () => import("../views/user/Posts.vue")
      },

    ]
  },
```
