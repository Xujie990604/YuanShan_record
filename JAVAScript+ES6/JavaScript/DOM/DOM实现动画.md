# DOM实现动画

## 间歇调用和超时调用

### window对象的方法

* 接收两个参数，第一个参数可以是字符串里面填写代码语句(就像是eval一样，把字符串当做代码来执行)，也可以是函数，不推荐使用字符串。
* 不推荐使用间歇调用，原因是后一个间歇调用可能在前一个间歇调用结束之前启动。
* 最佳模式是使用超时调用来模拟间歇调用。

### setTimeout(超时调用)

* 在多长时间后执行函数。
* setTimeout("test()",1000),setTimeout(test,1000)。第一个参数是字符串的时候使用"test()",第一个参数不是字符串的话直接放函数的引用test。
* 而且test()函数需要传递参数时，需要进行字符串的连接。

### setInterval(间歇调用)

* 每间隔多长时间就要执行一次函数。

### 为定时器赋予一个名字

* 调用之后会返回一个数值的ID。可以通过它来取消超时调用。
* 有些时候可以给正在进行一个动画(通过超时调用来控制的)的元素添加一个属性，以便于下一次进行动画时，用来清除上一次的动画以免上次未执行完毕的动画，影响当前动画的执行。
