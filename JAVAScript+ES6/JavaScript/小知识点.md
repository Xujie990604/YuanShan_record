# js中的小知识点

* alert（）接收的参数是字符串

* var 过的属性是不可配置的属性。无法delete掉。delete只能删除可配置的属性(对象的属性)。

* window.onload事件是在页面加载的时候调用。

* 不能在控制台中直接使用{}来表示空对象，控制台会把{}认为是块。而不是对象。需要({})这种形式让控制台强制认为{}是一个空对象。

* 逗号运算符 (a,b)先运算前面的表达式，然后运算后面的表达式。最后输出后面的表达式

```js
input.select()
document
```
