# 高性能JavaScript

* 优点：弱类型，灵活强大的函数，富有表现力的对象字面量表示法，无类别(class-free)的对象系统允许对象直接从其他对象继承属性
* 缺点：基于全局变量的编程模型(为什么？如何避免)
